# FINRA 2210 — Communications with the Public (Summary)

This document is a simplified summary for demo purposes.

Key ideas:
- Communications must be fair and balanced, not misleading.
- Promissory language and guarantees are high risk.
- Performance presentations must include appropriate context and disclosures.
- Retail communications often require principal pre-approval.
- Firms must supervise communications and maintain records.

Practical notes:
- Avoid absolute statements like "will" or "guaranteed" when discussing outcomes.
- Use risk disclosures near the claim, not buried.
- When referencing past performance, include material limitations.

Escalation:
- If a message contains performance claims, forward to compliance review.
- If a message appears to provide personalized advice, confirm suitability workflow.
